# A Big Thanks!

Building pgxcli has been a wild ride. Honestly, it all started when I was casually searching for a "postgres tutorial in tamil"—and somehow, one thing led to another, and I ended up building a full-blown Postgres CLI client!

This project wouldn't be possible without a whole bunch of amazing tools, projects, and people. So, I just wanted to take a moment to give a massive shoutout to the giants whose shoulders we stand on:

* **[pgcli](https://github.com/dbcli/pgcli)** - Proud to say we are heavily inspired by pgcli!
* **[pgx](https://github.com/jackc/pgx)** & **[Charm](https://github.com/charmbracelet)** - Without these two, pgxcli literally wouldn't exist. They are the core foundations of the pgxcli.
* **[CockroachDB](https://github.com/cockroachdb/cockroach)** - Thanks for being an incredible source of code references when I was figuring things out.
* **[sqls](https://github.com/sqls-server/sqls)** - Huge thanks for the sqls server code! Without it, pgxcli's context-aware completion would probably have been released sometime in 2030 hahaha!
* **[bubbline](https://github.com/knz/bubbline)** - This serves as the main interactive prompt component in pgxcli. It's an absolute game-changer.
* **[sjmignot](https://github.com/sjmignot)** - Massive thanks for implementing the syntax highlighting in bubbline on your [fork](https://github.com/chalk-ai/bubbline). It made a world of difference!
* **Our Contributors** - A big thank you to everyone who has contributed, stargazers and given feedback. You rock!

If I missed anyone, please know that your work is deeply appreciated. Now, let's get back to querying!
