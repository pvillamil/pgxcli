## Installation

Pick your operating system or preferred installation method:

<Tabs>
  <TabItem value="linux" label="Linux">
    Download the package for your distro from the [releases page](https://github.com/Balaji01-4D/pgxcli/releases).

    **Debian / Ubuntu**
    ```bash
    sudo dpkg -i pgxcli_*_linux_amd64.deb
    ```

    **Fedora / RHEL**
    ```bash
    sudo rpm -i pgxcli_*_linux_amd64.rpm
    ```

    **Alpine**
    ```bash
    sudo apk add --allow-untrusted pgxcli_*_linux_amd64.apk
    ```

    **Arch Linux**
    ```bash
    sudo pacman -U pgxcli_*_linux_amd64.pkg.tar.zst
    ```

    **Archive (tar.gz)**
    ```bash
    tar -xzf pgxcli_*_linux_amd64.tar.gz
    sudo mv pgxcli /usr/local/bin/
    ```
  </TabItem>
  <TabItem value="mac" label="Mac">
    Grab the latest `.tar.gz` for your architecture from the [releases page](https://github.com/Balaji01-4D/pgxcli/releases).

    **Manual Installation**
    ```bash
    # For Apple Silicon (M1/M2/M3)
    tar -xzf pgxcli_0.1.1_darwin_arm64.tar.gz
    sudo mv pgxcli /usr/local/bin/

    # For Intel Mac
    tar -xzf pgxcli_0.1.1_darwin_amd64.tar.gz
    sudo mv pgxcli /usr/local/bin/
    ```
  </TabItem>
  <TabItem value="windows" label="Windows">
    **Installer (MSI)** — recommended

    Download the `.msi` from the [releases page](https://github.com/Balaji01-4D/pgxcli/releases) and run it. pgxcli will be added to your `PATH` automatically.

    **Archive (zip)**

    Grab the `.zip` from the [releases page](https://github.com/Balaji01-4D/pgxcli/releases), extract it, and place `pgxcli.exe` anywhere on your `PATH`.
  </TabItem>
  <TabItem value="go-install" label="Go Install">
    Make sure you have **Go 1.21+**.

    ```bash
    go install github.com/balaji01-4d/pgxcli/cmd/pgxcli@latest
    ```
  </TabItem>
  <TabItem value="source" label="Source Code">
    Make sure you have **Go 1.21+**.

    ```bash
    git clone https://github.com/balaji01-4d/pgxcli.git
    cd pgxcli
    make build
    ```

    The binary lands in `bin/app`. Move it wherever you like (e.g., `/usr/local/bin/pgxcli`).
  </TabItem>
</Tabs>

## Quick Start

Connect to a database:

```bash
pgxcli mydb myuser
```

That's it. You'll get a REPL with syntax highlighting, autocompletion, and history — ready to go.

:::tip
On first run, pgxcli creates a config file at `~/.config/pgxcli/config.toml`. You can customize your prompt, theme, pager, and more. See [Configuration](/docs/guides/configuration).
:::

### Try a Query

```sql
SELECT version();
```

Results render in a clean table with execution time. If the output is long, pgxcli pages it automatically.

### Exit

Type `\q`.
