# pgxcli Documentation

This directory contains the documentation site for pgxcli, built with [Docusaurus](https://docusaurus.io/).

## 🚀 Project Structure

```
.
├── docs/                  # Markdown files for documentation
├── src/
│   ├── components/        # React components
│   ├── css/               # Custom CSS
│   └── pages/             # React pages
├── static/                # Static assets (images, favicon, robots.txt)
├── docusaurus.config.ts   # Site configuration
└── sidebars.ts            # Sidebar configuration
```

## 🧞 Commands

All commands are run from the root of the `docs` directory:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm start`               | Starts local dev server at `localhost:3000`      |
| `npm run build`           | Build your production site to `./build/`         |
